Teste Técnico - Estágio DevOps

Sobre

Este site apresenta, de maneira simplificada, o desenvolvimento do Teste Técnico para a vaga de Estágio DevOps na Ivory IT, apresentando as etapas do projeto, as tecno logias utilizadas e a implementação do CI/CD.

Processo de Desenvolvimento

O projeto segue um fluxo estruturado para garantir organização e eficiência. As etapas planejadas incluem:

1 - Desenvolvimento do site (Estruturação, layout e definição do conteúdo)

2 - Configuração do repositório (Inicialização no GitHub e organização dos arquivos)

3 - Versionamento com Git (Gerenciamento de commits e branches)

4 - Hospedagem na AWS (Publicação do site utilizando S3)

5 - Implementação do SonarCloud (Análise e qualidade do código)

6 - Implementação do CI/CD (Automação do fluxo de deploy)
